# 智编助手 SmartEdit AI - 产品功能说明

> 版本：v1.0.0 | 更新日期：2025年12月

## 产品概述

**智编助手 SmartEdit AI** 是一款专为微信公众号创作者打造的浏览器扩展工具，集成了 AI 写作、样式排版、素材管理、配图搜索等核心功能，帮助创作者提升内容生产效率。

### 目标用户
- 微信公众号运营者
- 内容创作者
- 新媒体编辑
- 自媒体从业者

### 核心价值
- **效率提升**：AI 辅助写作，减少创作时间
- **质量提升**：专业排版模板，提升文章视觉效果
- **素材管理**：本地存储 + 云端同步，内容不丢失
- **一站式服务**：写作、排版、配图、发布全流程覆盖

## 功能模块

### 1. 📝 排版样式库

#### 功能描述
提供丰富的预设样式模板，覆盖公众号文章常用的各类排版元素。

#### 样式分类

| 分类 | 数量 | 说明 |
|------|------|------|
| 标题样式 | 6+ | 简约标题、左侧装饰、编号标题、底色标题等 |
| 正文样式 | 4+ | 引用段落、重点提示、警告提示、成功提示 |
| 分割线 | 5+ | 简约线条、渐变线条、圆点分割、END分割等 |
| 卡片组件 | 2+ | 简约卡片、数据卡片 |
| 列表样式 | 2+ | 勾选列表、编号列表 |
| 引导关注 | 2+ | 关注引导、底部引导 |

#### 特色功能
- **一键插入**：点击样式即插入编辑器光标位置
- **实时预览**：鼠标悬停显示样式效果
- **主题换色**：10+ 预设主题色，支持自定义颜色
- **全文换色**：一键替换全文所有主题色

### 2. ✨ AI 写作助手

#### 功能描述
基于大语言模型的智能写作辅助工具，支持多种 AI 服务提供商。

#### 支持的 AI 服务

| 服务商 | 模型 | 特点 |
|--------|------|------|
| OpenAI | GPT-3.5/4 | 国际领先，效果最佳 |
| DeepSeek | deepseek-chat | 国产优质，性价比高 |
| 阿里云通义 | qwen-plus | 中文理解能力强 |
| 智谱 AI | GLM-4 | 国产大模型，免费额度 |
| Moonshot | moonshot-v1 | 长文本处理能力强 |
| 硅基流动 | Qwen2.5 | 开源模型，成本低 |
| 自定义 | - | 支持任意 OpenAI 兼容接口 |

#### 核心功能

##### 2.1 AI 标题生成
- **输入**：文章内容或主题关键词
- **输出**：10 个高点击率标题建议
- **特点**：
  - 多种标题风格（悬念式、数字式、疑问式等）
  - 一键复制使用
  - 支持重新生成

##### 2.2 AI 标题评分
- **输入**：待评估的标题
- **输出**：0-100 分评分 + 优化建议
- **评分维度**：
  - 吸引力
  - 清晰度
  - 情感共鸣
  - SEO 友好度

##### 2.3 AI 文章生成
- **写作模板**：
  - 🔥 热点解读
  - 📚 教程干货
  - 📖 故事叙述
  - 📋 清单盘点
  - ⭐ 测评推荐
  - 🎤 访谈对话
- **风格选择**：专业严谨 / 轻松活泼 / 情感细腻 / 幽默风趣
- **长度控制**：短文(500-800字) / 中等(1000-1500字) / 长文(2000-3000字)

##### 2.4 划词改写
选中文字后自动弹出工具栏，支持：
- **润色**：优化文字表达，提升可读性
- **扩写**：增加内容细节，丰富表达
- **缩写**：精简内容，突出重点
- **换语气**：改变表达风格（正式/轻松/幽默等）

### 5. 💾 本地存储

#### 功能描述
基于 IndexedDB 的本地存储系统，支持草稿保存和素材收藏。

#### 存储类型

##### 5.1 草稿箱
- **自动保存**：编辑内容自动保存
- **版本历史**：保留修改记录
- **标签分类**：自定义标签管理
- **快速恢复**：一键恢复到编辑器

##### 5.2 素材收藏
- **多类型支持**：文章、图片、标题、金句、模板
- **来源追溯**：记录素材来源和链接
- **标签管理**：灵活的标签系统
- **快速插入**：一键插入编辑器

#### 导出功能

| 格式 | 说明 |
|------|------|
| Markdown | 标准 Markdown 格式 |
| Obsidian 格式 | 包含 YAML frontmatter 和双链语法 |
| JSON 备份 | 完整数据备份，支持导入恢复 |

### 6. 💎 Obsidian 集成

#### 功能描述
通过 Obsidian Local REST API 插件，实现与 Obsidian 知识库的直接同步。

#### 功能特点
- **直接保存**：一键保存到 Obsidian Vault
- **自动格式化**：生成 YAML frontmatter
- **双链支持**：自动生成 `[[链接]]` 语法
- **路径配置**：自定义保存目录
- **连接状态**：实时显示连接状态

#### 保存格式示例
```markdown
---
title: 文章标题
created: 2025-12-09T10:00:00.000Z
type: draft
tags:
  - 公众号
  - 草稿
---

# 文章标题

文章内容...
```

## 技术规格

### 运行环境
- **浏览器**：Chrome 88+、Edge 88+
- **扩展标准**：Manifest V3
