## 快速开始

### 第一步：安装扩展

1. 下载扩展文件（zip 包或文件夹）
2. 打开 Chrome 浏览器，访问 `chrome://extensions/`
3. 开启右上角的「**开发者模式**」
4. 点击「**加载已解压的扩展程序**」
5. 选择扩展文件夹（包含 `manifest.json` 的目录）

### 第二步：基础配置

1. 点击浏览器工具栏的扩展图标
2. 点击「**设置**」进入配置页面
3. 在「**AI 配置**」中：
   - 选择 AI 服务提供商
   - 输入 API Key
   - 点击「测试连接」验证

### 第三步：开始使用

1. 打开 [微信公众平台](https://mp.weixin.qq.com)
2. 进入「**图文消息**」编辑页面
3. 页面左侧会出现悬浮工具栏
4. 点击工具栏打开侧边栏，开始使用

## 安装配置

### AI 服务配置

#### 推荐配置方案

| 方案 | 服务商 | 成本 | 适合人群 |
|------|--------|------|----------|
| 免费入门 | 智谱 AI | 免费额度 | 新手体验 |
| 性价比 | DeepSeek | 低成本 | 日常使用 |
| 专业版 | OpenAI | 按量付费 | 高质量需求 |

#### 配置步骤

**方案一：智谱 AI（推荐新手）**

1. 访问 [智谱 AI 开放平台](https://open.bigmodel.cn/)
2. 注册账号，获取免费额度
3. 在「API Keys」页面创建密钥
4. 复制 API Key 到扩展设置

**方案二：DeepSeek（推荐日常）**

1. 访问 [DeepSeek 平台](https://platform.deepseek.com/)
2. 注册并充值（价格低廉）
3. 创建 API Key
4. 复制到扩展设置

**方案三：OpenAI**

1. 访问 [OpenAI Platform](https://platform.openai.com/)
2. 注册账号并绑定支付方式
3. 创建 API Key
4. 复制到扩展设置

### 图片服务配置

#### Unsplash 配置

1. 访问 [Unsplash Developers](https://unsplash.com/developers)
2. 点击「Your apps」→「New Application」
3. 同意条款，创建应用
4. 复制「Access Key」到扩展设置

#### Pixabay 配置

1. 访问 [Pixabay](https://pixabay.com/)
2. 注册账号
3. 访问 [API 文档页](https://pixabay.com/api/docs/)
4. 复制页面显示的 API Key 到扩展设置

### Obsidian 配置（可选）

#### 前置要求
- 已安装 [Obsidian](https://obsidian.md/)
- 已安装 [Local REST API 插件](https://github.com/coddingtonbear/obsidian-local-rest-api)

#### 配置步骤

1. **安装插件**
   - 打开 Obsidian → 设置 → 第三方插件
   - 搜索「Local REST API」并安装
   - 启用插件

2. **获取 API Key**
   - 进入插件设置
   - 复制显示的 API Key

3. **配置扩展**
   - 进入扩展设置 → Obsidian
   - 启用 Obsidian 集成
   - 填写 API Key
   - 设置默认保存路径（如 `公众号/草稿`）
   - 点击「测试连接」验证

## 功能使用

### 📝 样式排版

#### 插入样式

1. 在编辑器中将光标定位到目标位置
2. 打开侧边栏 → 选择「📝 排版」
3. 浏览样式分类：标题 / 正文 / 分割 / 卡片 / 列表 / 引导
4. 点击样式即可插入

#### 一键换色

1. 在「排版」面板找到「🎨 主题色」区域
2. 点击预设颜色或输入自定义颜色
3. 点击「应用到全文」替换所有主题色

#### 全文排版

1. 在「排版」面板找到「📐 一键排版」
2. 选择排版模板：
   - 🌿 简约清新 - 适合日常分享
   - 💼 商务专业 - 适合职场内容
   - 🎨 文艺优雅 - 适合情感文章
   - 🚀 科技现代 - 适合科技资讯
3. 点击应用

---

### ✨ AI 写作

#### 生成标题

1. 打开侧边栏 → 选择「✨ 写作」
2. 在「标题生成」区域粘贴文章内容
3. 点击「生成标题」
4. 从 10 个建议中选择合适的标题
5. 点击标题即可复制

#### 标题评分

1. 在「标题评分」区域输入标题
2. 点击「评分」
3. 查看评分结果和优化建议

#### 生成文章

1. 切换到「文章生成」标签
2. 选择写作模板（热点解读/教程干货/故事叙述等）
3. 输入文章主题或关键词
4. 选择风格和长度
5. 点击「生成文章」
6. 等待生成完成，点击「插入编辑器」

#### 划词改写

1. 在编辑器中选中一段文字
2. 自动弹出工具栏
3. 选择操作：
   - **润色** - 优化表达，提升可读性
   - **扩写** - 增加细节，丰富内容
   - **缩写** - 精简内容，突出重点
   - **换语气** - 改变风格（正式/轻松/幽默）
4. 等待 AI 处理，结果自动替换原文

### 📰 RSS 订阅

#### 浏览文章

1. 打开侧边栏 → 选择「📰 RSS」
2. 点击「刷新」获取最新文章
3. 点击文章标题查看详情
4. 点击「查看原文」跳转到源站

#### 添加订阅源

1. 进入扩展设置 → RSS 订阅
2. 点击「添加订阅源」
3. 填写：
   - 名称：订阅源显示名称
   - URL：RSS 地址
   - 分类：自定义分类
4. 点击保存

#### 收藏文章

1. 在 RSS 文章详情页
2. 点击「⭐ 收藏文章」
3. 文章将保存到「存储 → 收藏」

---

### 💾 本地存储

#### 保存草稿

1. 打开侧边栏 → 选择「💾 存储」
2. 在「草稿」标签点击「保存当前内容」
3. 编辑器内容将自动保存

#### 管理草稿

1. 在草稿列表点击草稿查看详情
2. 可执行操作：
   - 📝 插入编辑器
   - 📄 导出 MD
   - 📋 复制内容
   - 💎 保存到 Obsidian
   - 🗑️ 删除

#### 素材收藏

收藏来源：
- RSS 文章 → 点击「收藏」
- 编辑器选中内容 → 右键「收藏到素材库」
- 手动添加

#### 导出数据

1. 切换到「📤 导出」标签
2. 勾选要导出的内容
3. 选择导出方式：
   - **合并导出 Markdown** - 所有内容合并为一个文件
   - **Obsidian 格式** - 包含 YAML frontmatter
   - **逐个导出** - 每条内容单独一个文件
   - **直接保存到 Obsidian** - 通过 API 直接写入

#### 备份恢复

**导出备份：**
1. 在导出标签点击「📦 导出全部数据 (JSON 备份)」
2. 保存 JSON 文件

**导入恢复：**
1. 在导出标签点击「📥 导入数据」
2. 选择之前导出的 JSON 文件
3. 确认导入

### 🖼️ 配图搜索

#### 搜索图片

1. 打开侧边栏 → 选择「🖼️ 配图」
2. 在搜索框输入关键词（支持中英文）
3. 或点击快捷标签：风景、科技、商务、美食等

1. 点击图片查看大图预览
2. 点击「插入编辑器」
3. 图片将插入到编辑器光标位置

- 点击「Unsplash」或「Pixabay」切换图片来源
- 不同图片源风格不同，可根据需要选择

#### 清除格式

1. 选中需要清除格式的内容
2. 打开侧边栏 → 选择「🔧 工具」
3. 点击「清除格式」

1. 打开侧边栏 → 选择「📋 MD」
2. 在输入框粘贴 Markdown 内容
3. 点击「转换」

## 常见问题

### Q1: AI 功能无法使用？

**可能原因：**
1. 未配置 API Key
2. API Key 无效或过期
3. 余额不足
4. 网络问题

**解决方法：**
1. 检查设置中的 API Key 是否正确
2. 点击「测试连接」验证
3. 登录对应平台检查余额
4. 尝试使用代理

### Q2: 图片搜索没有结果？

**可能原因：**
1. 未配置图片服务 API Key
2. 搜索关键词过于特殊
3. 网络问题

**解决方法：**
1. 在设置中配置 Unsplash/Pixabay API Key
2. 尝试更通用的关键词
3. 使用英文关键词效果更好

### Q3: 样式插入后颜色不对？

**解决方法：**
1. 使用「一键换色」功能
2. 选择目标主题色
3. 点击「应用到全文」

### Q4: Obsidian 连接失败？

**检查项：**
1. Obsidian 是否正在运行
2. Local REST API 插件是否启用
3. API Key 是否正确
4. 端口是否为默认 27124

**解决方法：**
1. 确保 Obsidian 已打开
2. 在 Obsidian 设置中确认插件已启用
3. 重新复制 API Key
4. 检查防火墙设置

### Q5: RSS 无法获取内容？

**可能原因：**
1. RSS 源地址无效
2. 跨域限制
3. 网站不支持 RSS

**解决方法：**
1. 验证 RSS 地址是否正确
2. 尝试其他 RSS 源
3. 启用代理设置

### Q6: 数据丢失怎么办？

**预防措施：**
1. 定期使用「导出全部数据」备份
2. 启用 Obsidian 自动同步

**恢复方法：**
1. 使用「导入数据」恢复备份
2. 从 Obsidian 重新获取
