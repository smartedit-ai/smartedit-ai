## 版本历史

### v1.0.0 (2025-12)

**新功能**
- 样式库：6 类 20+ 预设样式
- AI 写作：标题生成、评分、文章生成、划词改写
- 配图中心：Unsplash + Pixabay 集成
- RSS 订阅：订阅管理、文章收藏
- 本地存储：IndexedDB 草稿和收藏
- Obsidian 集成：Local REST API 同步
- 热点搜索：Tavily API 集成
- 实用工具：格式清除、缩进、排版
