# 更新日志 Changelog

本项目遵循 [语义化版本](https://semver.org/lang/zh-CN/) 规范。

---

## [0.0.1] - 2025-12-06

### 🎉 首次发布

这是智编助手的首个版本，包含以下核心功能：

### ✨ 新增功能

#### 🎨 超级编辑器
- **侧边栏样式库** - 丰富的标题、正文、分割线、卡片、列表样式模板
- **一键换色系统** - 10+ 预设主题色，支持自定义颜色
- **全文一键排版** - 4 套精选模板（简约清新/商务专业/文艺优雅/科技现代）
- **快捷格式工具** - 清除格式、首行缩进、调整行高

#### 🤖 AI 创作引擎
- **AI 标题生成** - 基于文章内容，一键生成 10 个高点击率标题
- **AI 标题评分** - 对标题进行 0-100 分评分，提供优化建议
- **AI 文章生成** - 选择标题后自动生成完整文章
- **AI 划词改写** - 选中文字即可润色、扩写、缩写
- **一键插入编辑器** - 生成的内容可直接插入微信编辑器

#### 🔌 多 AI 服务商支持
- **OpenAI** - GPT-3.5 / GPT-4 / GPT-4o
- **DeepSeek** - DeepSeek-V3 / DeepSeek-Chat
- **阿里云百炼** - Qwen-Max / Qwen-Plus / Qwen-Turbo
- **硅基流动 SiliconFlow** - 多模型支持
- **月之暗面 Moonshot** - Kimi 系列模型
- **智谱 AI** - GLM-4 / GLM-4-Flash
- **自定义配置** - 支持任意 OpenAI 兼容 API

#### 🖼️ 配图中心
- **Unsplash 集成** - 海量高质量免费图片
- **Pixabay 集成** - 丰富的免版权图片素材
- **背景模板** - 预设渐变背景和纯色背景
- **一键插入** - 搜索图片后直接插入编辑器

#### 🛠️ 便捷工具
- **悬浮工具栏** - 页面左侧快捷入口
- **划词工具栏** - 选中文字自动弹出 AI 工具
- **右键菜单** - 快速调用 AI 改写功能
- **快捷键支持** - Ctrl+B 加粗、Ctrl+I 斜体等

### 🔧 技术特性
- 基于 Chrome Manifest V3 标准
- 使用 React 18 + TypeScript + TailwindCSS 构建
- Vite 构建工具，支持热更新开发
- 本地存储 API Key，保护用户隐私
